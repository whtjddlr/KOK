
  # KoK

  This is a code bundle for KoK. The original project is available at https://www.figma.com/design/KFqVg7C702zZ10sPCv0r8n/Random-Meeting-Place-App.

  ## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.

  ## Supabase sync

  For online ready and redraw voting, run `supabase/ready-vote-sync.sql` in the Supabase SQL Editor.
